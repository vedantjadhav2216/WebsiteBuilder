# 📖 About

AI Website Builder is a Full Stack MERN application that enables users to create complete websites simply by describing what they want in natural language.

The application uses Artificial Intelligence through the OpenRouter API to generate clean HTML, CSS, and JavaScript code. Users can edit the generated code inside a professional code editor, preview the website in real time, save projects, and manage them from a personalized dashboard.

# ⚙ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/vedantjadhav2216/AI-Website-Builder.git
```

```bash
cd AI-Website-Builder
```

---

## 2️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

## 3️⃣ Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

# 🔑 Environment Variables

## Backend (.env)

```env
PORT=5000

MONGODB_URI=YOUR_MONGODB_CONNECTION_STRING

JWT_SECRET=YOUR_SECRET_KEY

OPENROUTER_API_KEY=YOUR_OPENROUTER_API_KEY

RAZORPAY_KEY_ID=YOUR_RAZORPAY_KEY_ID

RAZORPAY_KEY_SECRET=YOUR_RAZORPAY_KEY_SECRET
```

---

## Frontend (.env)

```env
VITE_BACKEND_URL=http://localhost:5000
```

If Google Authentication is enabled, configure your Firebase credentials in the frontend configuration.

# ⚙ How It Works

### Step 1

User creates an account or logs in.

↓

### Step 2

User enters a website prompt.

↓

### Step 3

Backend sends the prompt to OpenRouter AI.

↓

### Step 4

AI generates HTML, CSS and JavaScript.

↓

### Step 5

Generated code opens inside Monaco Editor.

↓

### Step 6

User edits the code if required.

↓

### Step 7

Live Preview updates instantly.

↓

### Step 8

Website is saved into MongoDB.

↓

### Step 9

Projects are managed through Dashboard.
